# ClipForge AI

Turn long videos (podcasts, calls, streams, webinars) into viral short-form clips automatically.

## Run

    npm install
    npm run dev

Opens at http://localhost:3000.

## Marketing routes
/  /pricing  /blog  /blog/[slug]  /login  /signup  /contact  /compliance  /changelog

## App routes
/dashboard  /projects  /projects/[id]  /clips  /clips/[id]  /accounts  /schedule  /templates  /team  /analytics  /settings
